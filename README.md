# 📊 Анализ клиентских транзакций (SQL)

## 📌 Задача
Проанализировать транзакции клиентов за период 01.06.2015 – 01.06.2016 и выявить поведение клиентов, динамику продаж и ключевые метрики.

## 🛠 Инструменты
- SQL (MySQL)
- CTE (WITH)
- Агрегатные функции (SUM, AVG, COUNT)
- DATE_FORMAT
- JOIN

## 📊 Что сделано

### 🔹 Подготовка данных
- Очистка данных (замена пустых значений на NULL)
- Приведение типов данных (Age → INT)
- Загрузка данных из CSV

### 🔹 Анализ клиентов
- Найдены клиенты с непрерывной активностью (12 месяцев подряд)
- Рассчитаны:
  - средний чек
  - средние траты в месяц
  - количество операций

### 🔹 Анализ по месяцам
- Средний чек по месяцам
- Количество операций
- Количество активных клиентов

### 🔹 Доли и метрики
- Доля операций от общего количества за год
- Доля выручки по месяцам

### 🔹 Сегментация клиентов
- Анализ по полу (M / F / NA)
- Доля операций и затрат по каждой группе

## 💡 Результаты (инсайты)
- Выявлены клиенты с высокой лояльностью (регулярные покупки)
- Определены месяцы с наибольшей активностью
- Найдены различия в поведении клиентов по полу
- Выделены ключевые факторы, влияющие на выручку

## 📎 Данные
Учебный датасет (транзакции и клиенты)
